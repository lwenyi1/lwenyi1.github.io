# School Projects


## CG2111A: "Alex to the Rescue"
- [GitHub repo](https://github.com/lwenyi1/CG2111A-Alex)
- Took this course in AY23/24 Sem 2, where we built a robot with movement, LIDAR, colour detection, ultrasonic distance measurement and remote-control capabilities using various hardware and software to navigate a maze and complete tasks as part of the NUS CG2111A module.
- My contributions:
  - Set up ROS Noetic, RPLidar and SLAM on a Raspberry Pi and Laptop for mapping. Programmed the network connection software and TLS on the Pi and Laptop for remote-control.
  - Programmed the firmware for the motors, ultrasonic sensor and colour sensor, including the colour detection algorithm, on an Arduino Mega and set up the UART connection between the Arduino and the Pi.
  - Wired the motors, sensors and other additional components.
- I enjoyed this project so much I went on to TA for this course the year after 🤯.
