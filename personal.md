# Personal Projects

## Vimdows
- [GitHub repo](https://github.com/lwenyi1/Vimdows)
- I was using my tr*ckpad on MS Word one day feeling like a total loser when I realised wouldn't it be cool if I could use 'h', 'j', 'k' and 'l' to move around instead.
  The natural thing to do was implement a program that sits between the user space and keyboard driver on Windows, catching keypresses and remapping them to emulate Vim movement and editing keybinds.
- Created the base program engine which intercepts keypresses, processes them and outputs the mapping.
- Implemented the Vim “state machine” to emulate Normal and Insert modes and handles the processing of keys to turn them into Vim keybind equivalents.
